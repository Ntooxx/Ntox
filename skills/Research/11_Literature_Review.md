# NTOX — Literature Review Engine

## Purpose

Systematic analysis of existing knowledge. Not just reading — critical evaluation, synthesis, and identification of gaps. The foundation on which new research is built.

## Core Principle

> Before you can advance knowledge, you must understand what knowledge exists.
> Before you can understand it, you must evaluate it.
> Before you can evaluate it, you must find it.

## Literature Review Framework

### Phase 1: Scope Definition

Before reviewing, define:
1. **Research question**: What am I trying to understand?
2. **Scope**: What fields are relevant?
3. **Time frame**: How far back should I look?
4. **Inclusion criteria**: What counts as relevant?
5. **Exclusion criteria**: What should be ignored?

### Phase 2: Systematic Search

Search strategy:
- **Core terms**: Primary keywords for the topic
- **Synonyms**: Alternative terms for the same concepts
- **Related fields**: Adjacent disciplines
- **Methodological terms**: Specific methods used
- **Negative terms**: What to exclude

Search structure:
```
(topic OR synonym1 OR synonym2)
AND
(field1 OR field2 OR field3)
AND
(method1 OR method2)
NOT
(exclusion1 OR exclusion2)
```

### Phase 3: Quality Assessment

For each source, evaluate:

| Criterion | Questions |
|-----------|-----------|
| **Authority** | Who wrote it? What are their credentials? |
| **Currency** | When was it published? Is it still relevant? |
| **Objectivity** | Is there bias? Conflict of interest? |
| **Methodology** | Are methods sound? Sample adequate? |
| **Relevance** | Does it address the research question? |
| **Replicability** | Can the results be reproduced? |
| **Impact** | Has it been cited? Influenced the field? |

### Phase 4: Analysis and Synthesis

After collecting sources:

1. **Categorize**: Group by theme, method, or finding
2. **Compare**: What do sources agree on? Disagree on?
3. **Evaluate**: Which findings are most reliable?
4. **Synthesize**: What is the overall picture?
5. **Identify gaps**: What hasn't been studied?
6. **Identify trends**: Where is the field moving?

### Phase 5: Gap Analysis

Types of gaps:

| Gap Type | Description |
|----------|-------------|
| **Empirical gap** | Data doesn't exist |
| **Methodological gap** | No method to study this |
| **Theoretical gap** | No theory explains this |
| **Conceptual gap** | No framework to think about this |
| **Integration gap** | Knowledge exists in silos |
| **Temporal gap** | Studies are outdated |

### Phase 6: Output

Produce:
- Summary of current knowledge
- Map of agreements and disagreements
- Assessment of evidence quality
- Identification of gaps
- Recommendations for future research
- Key papers and their contributions

## Critical Reading Framework

For any paper or source:

1. **Abstract/Summary**: What is claimed?
2. **Introduction**: Why does this matter?
3. **Methods**: How was this studied?
4. **Results**: What was found?
5. **Discussion**: What does it mean?
6. **Limitations**: What are the weaknesses?
7. **Conclusion**: What is the take-away?
8. **References**: What led to this?

For each section, ask:
- Is this justified?
- Are there alternatives?
- What is missing?
- Do I agree? Why or why not?

## Common Literature Review Errors

| Error | Description | Prevention |
|-------|-------------|------------|
| **Cherry-picking** | Only selecting supporting evidence | Search for disconfirming evidence |
| **Recency bias** | Over-weighting recent work | Include foundational work |
| **Citation bias** | Only citing high-profile sources | Include diverse sources |
| **Methodological bias** | Favoring certain methods | Include multiple methodological approaches |
| **Confirmation bias** | Interpreting evidence to support beliefs | Actively seek contradicting evidence |
| **Scope creep** | Expanding beyond useful bounds | Stick to the research question |
| **Missing key work** | Overlooking important contributions | Systematic search strategy |

## Integration With Other Modules

- **Research Engine**: Literature review is a core research activity
- **Critical Thinking**: Provides tools for evaluating sources
- **Scientific Method**: Literature provides context for hypothesis generation
- **Discovery Engine**: Gaps in literature are discovery opportunities
- **Meta Cognition**: Monitors for biases in the review process
- **Memory**: Literature knowledge is stored and retrieved as needed

## The Literature Review Mantra

> I stand on the shoulders of giants.
> But I must first find them.
> And I must check that they are standing on solid ground.
> And I must see what they are not looking at.
> Then I can see further.
