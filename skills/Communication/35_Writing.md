# NTOX — Writing Engine

## Purpose

The art of expressing ideas in text. Writing is thinking made visible. If you can write it clearly, you understand it clearly. If you can't write it clearly, you don't understand it well enough.

## Core Principle

> Writing is thinking made visible.
> Clear writing requires clear thinking.
> If you can't write it clearly, you don't understand it well enough.
> Revision is not a sign of weakness.
> It is a sign of commitment to clarity.

## Writing Framework

### Phase 1: Planning
- What is the purpose of this piece?
- Who is the audience?
- What is the key message?
- What is the structure?
- What evidence will be used?

### Phase 2: Drafting
- Get ideas down first
- Don't worry about perfection
- Focus on structure and content
- Use clear, simple language
- Write in your voice

### Phase 3: Revising
- Check structure and flow
- Check clarity and precision
- Check evidence and support
- Check tone and voice
- Cut unnecessary words

### Phase 4: Editing
- Check grammar and punctuation
- Check spelling and formatting
- Check consistency
- Check citations
- Final polish

### Phase 5: Publishing
- Choose appropriate format
- Consider presentation
- Test readability
- Get feedback
- Iterate

## Writing Principles

### 1. Clarity
- Use simple language
- Avoid jargon
- Define terms
- Use active voice
- One idea per sentence

### 2. Structure
- Start with the conclusion
- Use clear headings
- Use transitions
- End with a summary
- Guide the reader

### 3. Evidence
- Back up claims with data
- Use specific examples
- Cite sources
- Acknowledge limitations
- Be transparent

### 4. Voice
- Write in your voice
- Be authentic
- Be confident
- Be humble
- Be engaging

### 5. Revision
- Revise multiple times
- Cut unnecessary words
- Simplify complex sentences
- Strengthen weak arguments
- Polish the final version

## Common Writing Errors

| Error | Description | Prevention |
|-------|-------------|------------|
| **Jargon** | Too much technical language | Use plain language |
| **Wordiness** | Too many words | Cut ruthlessly |
| **Passive voice** | Unclear who is acting | Use active voice |
| **No structure** | Disorganized | Use clear structure |
| **No evidence** | Claims without support | Back up with data |
| **No audience awareness** | Wrong level for audience | Analyze audience |
