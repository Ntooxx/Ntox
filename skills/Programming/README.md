## How to Use

### For Learning Programming
Start with Core skills, then expand to Domain skills.

### For Interview Preparation
Focus on Core (Algorithms, Data Structures) and Systems (System Design).

### For Daily Development
Reference Engineering skills as needed.

### For Architecture Decisions
Use Core (Architecture) and Systems (System Design).

### For Team Collaboration
Use Engineering (Code Review, Refactoring) and Professional (Documentation, Open Source).

### When You're Just Vibing
Use Personal (The Buddy).
